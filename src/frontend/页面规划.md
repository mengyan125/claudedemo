# 学生反馈系统 - 前端页面规划

## 开发流程

每个页面严格按以下流程推进：

```
原型回顾 → Pencil 设计稿 → 截图确认 → 生成 Vue 代码 → 测试验证
```

- 使用 Pencil 工具生成页面设计稿（.pen 文件）
- 设计完成后截图给用户确认布局、配色、交互
- 确认无误后生成 Vue 3 + Element Plus 页面代码
- 代码遵循 CLAUDE.md 编码规范
