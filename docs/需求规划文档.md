# 学生反馈系统 - 需求规划文档

## 一、项目背景

加强教学日常管理，学生的反馈是其中重要一个环节，需要给学生建立反馈的通道，让学生可以随时对教学、管理等方面提出意见和建议。

## 二、项目目标

- 建立学生与学校管理层之间的沟通渠道
- 保护学生隐私，支持匿名反馈
- 及时发现和解决教学管理中的问题
- 为学校管理决策提供数据支持

## 三、角色定义

| 角色 | 权限范围 |
|------|----------|
| 学生 | 提交反馈、跟踪反馈、查看回复 |
| 系统管理员 | 全类别反馈查看/回复 + 统计 + 列表维护（类别管理、快捷回复管理等） |
| 角色管理员 | 对其他角色授权，通过创建用户选择职位和类型（学生/教师），支持单个或批量添加 |
| 类别管理员 | 只能查看被授权类别的反馈并回复 |

## 四、模块1：Web端学生身份

### 4.1 我要反馈

**表单字段：**

| 字段 | 规则 |
|------|------|
| 反馈类别 | 必选，下拉选择 |
| 与任课老师相关 | 单选框，仅当类别为教学相关（后台配置标记）时显示 |
| 反馈对象 | 选中"与任课老师相关"后显示，必填，范围为该学生**本学期任课教师** |
| 反馈主题 | ≤20字，必填 |
| 反馈内容 | ≤500字，必填 |
| 匿名/实名 | 单选 |
| 附件上传 | 最多9个文件；图片：jpg/png/gif；视频：mp4/avi/mov；单个≤25M |

**操作按钮：**
- **提交** → 跳转成功提示页
- **暂存** → 保存到反馈跟踪列表，再次打开显示上次编辑内容

### 4.2 反馈跟踪

**空状态：** "亲爱的同学，你还没有提过反馈意见哦。"

**有数据时：**
- 列表展示，按系统提交时间倒序排列
- 列表上方显示每种类别的数据条数
- 默认显示全部反馈，可按类别筛选
- 默认显示10条，点击"加载更多"下拉查看
- 支持关键词手动检索（不限字数字符，不做校验）

**列表项展示：**
- 标题（学生撰写的主题）
- 反馈类别
- 被反馈教师（关联任课关系时显示）
- 反馈时间
- 内容首句15字 + ……

**交互：**
- 选中列表项查看反馈详情
- 收到回复后新增**未读角标**，标题字体**加粗**提示

### 4.3 反馈详情

- 有回复：展示管理员回复内容，学生可追加回复
- 暂无回复：提示暂无回复

## 五、模块2：Web端管理员身份

### 5.1 反馈查看

**三个 Tab：**

| Tab | 内容 |
|-----|------|
| 全部 | 展示全部反馈列表 |
| 我的收藏 | 展示已收藏的反馈 |
| 提醒关注 | 收到的提醒消息 |

**筛选项：**

| 序号 | 筛选项 | 类型 |
|------|--------|------|
| 1 | 反馈类别 | 下拉选择 |
| 2 | 年级 | 下拉选择 |
| 3 | 班级 | 下拉选择，联动年级 |
| 4 | 反馈对象 | 选择/搜索教师 |
| 5 | 关键词 | 文本输入，模糊搜索 |
| 6 | 时间 | 时间范围选择 |
| 7 | 状态 | 下拉选择 |
| 8 | 回复情况 | 已回复/未回复 |

### 5.2 反馈详情

**内容展示：**
- 反馈类别
- 年级 / 班级 / 反馈对象（仅教学相关类别时展示）
- 反馈人（匿名则显示匿名）
- 反馈时间
- 反馈主题
- 反馈内容
- 附件（图片：弹窗预览；视频/其他：下载或浏览器预览）
- 是否收藏（可执行收藏/取消收藏）
- 回复状态（已回复/未回复）

**详情页两个 Tab：**

**反馈回复：**
- 两种方式：自己输入 / 快捷回复
- 快捷回复选中后填入编辑框，可修改
- 提交后回复数据动态显示在 Tab 下
- 可追加回复
- 回复内容 ≤ 500字

**备注提醒：**
- 可选择提醒的人（点击按钮弹出人员列表）
- 不选则默认关联人员都可见
- 备注内容 ≤ 500字

### 5.3 反馈统计

- 本日收到的反馈数量
- 本月收到的反馈数量
- 反馈最多的类别及条数
- 反馈类别占比（图表）
- 本学期反馈数量统计
- 任课教师被反馈前十统计（点击数据可进入教师被反馈列表）

### 5.4 教师被反馈列表

- 展示该教师被反馈的每条详情

### 5.5 反馈管理（类别管理）

| 操作 | 规则 |
|------|------|
| 新增 | 类别名称（≤10字，汉字/字母/数字）、是否关联任课教师、显示顺序 |
| 编辑 | 类别下有数据时**不可编辑** |
| 删除 | 类别下有数据时**不可删除** |
| 启用/停用 | 控制类别是否对学生可见 |

### 5.6 快捷回复管理（新增页面）

- 管理员维护快捷回复模板
- 支持增删改

## 六、模块3：角色设置

### 6.1 角色权限设置

- 管理角色（系统管理员、角色管理员、类别管理员）
- 配置角色可访问的功能模块

### 6.2 类别权限配置

- 按反馈类别分配查看权限
- 权限分配：为某类别添加有权限的人员
- 移除：取消某人对某类别的查看权限

### 6.3 用户管理

- 通过创建用户选择对应职位和类型（学生/教师）
- 支持单个添加或批量添加用户进行授权

## 七、字段校验规则汇总

| 字段 | 上限 | 格式 | 必填 | 校验时机 | 提示信息 |
|------|------|------|------|----------|----------|
| 反馈类别 | - | 选择 | 是 | 提交时 | 请选择反馈类别 |
| 反馈对象 | - | 选择 | 是（关联时） | 提交时 | 请选择任课教师 |
| 反馈主题 | 20字 | 汉字/字母/数字/符号 | 是 | 输入时+提交时 | 请输入反馈主题 |
| 反馈内容 | 500字 | 汉字/字母/数字/符号 | 是 | 输入时+提交时 | 请输入反馈内容 |
| 反馈回复 | 500字 | 汉字/字母/数字/符号 | 否 | 输入时 | 超出无法输入 |
| 备注提醒 | 500字 | 汉字/字母/数字/符号 | 否 | 输入时 | 超出无法输入 |
| 类别名称 | 10字 | 汉字/字母/数字 | 是 | 提交时+输入时 | 请输入不超过10个字符 / 该类别已存在 |
| 是否关联任课教师 | - | 选择 | 是 | 提交时 | 请选择是否关联任课教师 |
| 显示顺序 | 1位 | 数字 | 是 | 提交时+输入时 | 请输入数字 |
